== Description ==

* **[WordPress Pro plugins](http://web-profile.net/wordpress/plugins/ "WordPress Pro plugins")**
* **[Content-Grid](http://web-profile.net/wordpress/plugins/content-grid/ "Plugin page")**
* **[Donate](http://web-profile.net/donate/ "Support the development")**


Plugin adds 6 content zones and visual content-grid structure builder with 3 rows and 2 cols in each row for every page.
Each content zone has visual editor.
The Content-Grid structure is responsive.
All the plugin's data stored in postmeta table and primary content zone is not affected.
Check out [screenshots](https://wordpress.org/plugins/content-grid/screenshots/ "screenshots") to see how it works.


= Useful: =
* ["Page-list" - show list of pages with shortcodes](http://wordpress.org/plugins/page-list/ "list of pages with shortcodes")
* ["Iframe" - embed content](http://wordpress.org/plugins/iframe/ "embed content")
* [WordPress Pro plugins](http://web-profile.net/wordpress/plugins/ "WordPress Pro plugins")

== Installation ==

1. install and activate the plugin on the Plugins page
2. set custom content structure for each page
